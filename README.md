# 🛍️ FabricMuse – E-Commerce Frontend

A modern, responsive e-commerce frontend with cart functionality, filtering, and clean UI design.

---

## 🚀 Project Overview

**FabricMuse** is a modern, responsive e-commerce frontend designed to simulate a real-world online shopping experience. The project focuses on delivering a clean and intuitive user interface, smooth navigation, and practical shopping interactions using core web technologies.

Built with a strong emphasis on fundamentals, FabricMuse leverages **HTML, CSS, Bootstrap, and JavaScript** to implement dynamic features such as product filtering, cart management, and interactive UI behavior. It serves as a scalable base that can be extended into a full-stack application with backend integration in the future.

---

## ✨ Features

FabricMuse provides a complete browsing experience through multiple pages, including Home, Products, About, Services, Contact, and Authentication interfaces. Each page is designed with consistency and usability in mind, ensuring a seamless experience across devices.

Users can explore products through a responsive grid layout enhanced with category filtering, price filtering, and search functionality, making product discovery simple and efficient.

The shopping cart system allows users to add items, remove them, adjust quantities, and view dynamically updated totals. Cart data is managed within the browser to maintain continuity even after refreshing the page.

The interface is fully responsive and optimized for mobile, tablet, and desktop devices. Additional enhancements such as dark mode, smooth transitions, and a mobile-friendly navigation system contribute to a polished and modern user experience.
